# Shabbir Kataleri - AI & ML Developer Portfolio 

A modern, highly interactive, and visually stunning 3D portfolio website showcasing my skills, projects, and experiences in Artificial Intelligence, Machine Learning, and Blockchain Security.

## 🌟 Live Demo
https://shabbir-portfolio-omega.vercel.app

## 🛠️ Built With
- **React.js** (Frontend UI)
- **TypeScript** (Type Safety)
- **React Three Fiber & Drei** (3D Rendering & Models)
- **GSAP (GreenSock)** (Advanced scroll animations and transitions)
- **Vite** (Build Tool)

## ⚡ Features
- **Immersive 3D Experience**: A fully interactive 3D character modeled in the hero section that tracks your cursor.
- **Smooth Scrolling**: Implemented using GSAP ScrollSmoother.
- **Dynamic Projects Showcase**: A custom horizontal carousel to display AI/ML projects with rich dashbaord imagery.
- **Responsive Design**: Carefully crafted to look perfect on Mobile, Tablet, and Desktop screens.
- **Modern UI Elements**: Custom cursor routing, magnetic hover links, and glowing text aesthetics.

## 🚀 Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/Shabbir05/shabbir-portfolio.git
   cd shabbir-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
